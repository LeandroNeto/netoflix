# Netoflix
Prepare a pipoca, sente-se no sofá e dê o play! 😉🍿 <img align="right" alt="Dê o play!" height="400" width="250" src="https://acegif.com/wp-content/uploads/gif-eating-popcorn-15.gif"/>
